# 최신 변경사항

- 반려동물을 전신이 아닌 자연스러운 상반신 구도로 생성합니다.
- 최종 결과 비율을 4:5에서 5:7로 변경했습니다.
- 최종 파일을 2500×3500px JPG로 제공합니다.
- 기존 `contain` 방식의 좌우 배경 덧붙이기를 제거했습니다.
- GPT Image가 생성한 좌우 배경을 그대로 유지합니다.
- 1024×1536 생성 결과에서 위아래만 소량 중앙 크롭해 5:7로 만듭니다.
- 배경이 좌우에 이어 붙인 것처럼 보이지 않도록 프롬프트를 강화했습니다.
- `gpt-image-2`에서 오류를 발생시킨 `input_fidelity` 요청값은 사용하지 않습니다.
- 실제 입력·출력 토큰과 달러 비용 로그 기능을 유지합니다.
- 폰트, 글자 크기, 색상, 가로·세로 방향, 위치 편집 기능을 유지합니다.
